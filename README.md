# GreenAxis

Company website for **GreenAxis** — software development.

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **React 18**

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command   | Description           |
| --------- | --------------------- |
| `npm run dev`   | Start dev server      |
| `npm run build` | Production build      |
| `npm run start` | Run production server |
| `npm run lint`  | Run ESLint            |

## Project structure

- `src/app/` — App Router pages and layout
- `src/components/` — Reusable UI (Header, Footer, sections)
- `tailwind.config.ts` — Tailwind theme (GreenAxis palette)

## Deploy

Build and run in production, or deploy to [Vercel](https://vercel.com) by connecting this repo.
